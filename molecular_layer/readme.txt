SUPPLEMENTARY FILE S2

This folder contains MATLAB scripts for simulating the molecular layer of the model, 
as described in Step 2 of Supplementary Section S5.

Contents

Scripts:
- main.m: Simulates the dynamics of the molecular layer in a dendritic cell following 
  administration of the BNT162b2 vaccine.

Functions:
- model_equations_subcellular.m: Defines the equations for the molecular layer.
- parameters_subcellular.m: Contains parameter values for the molecular layer.
- simulation_DC.m: Integrates the system of ODEs defined in the molecular layer.

Text Files:
- README.txt: This file, providing an overview of folder contents.
